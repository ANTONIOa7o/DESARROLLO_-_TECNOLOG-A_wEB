Tipos de Variables

string
Cadena de texto

Texto o conjunto de caracteres, asi como cualquier referencia tipográfica

Sintaxis: dentro de comillas simples ' ' o comillas dobles " "
number
Números

valor de la variable que guarda indistintamente cualquier numero real

Sintaxis: valor del numero
boolean
buleanos

variables logicas, true=1 y false=0, serán requeridas para poder realizar comparaciones o permisos

Sintaxis:true o false
objets
objetos

Una representacion de las caracteristicas del elemento

Sintaxis:var nombreVariable = new NombreVariable()
null
nulo

la variable existe, es decir declarada pero sin valor. No accede, no encuentra o no define valor.

Sintaxis:var nombreVariable=nulll
undefined
No definida

No se podria acceder (no la encuentra) al entorno donde se encuentra la variable

Sintaxis:var nombreVariable = undefined
