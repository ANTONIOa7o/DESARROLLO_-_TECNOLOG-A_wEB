ESTRUCTURAS DE CONTROL

Un conjunto de acciones que nos permiten acceder a la posibilidad de poder
tomar decisiones (ejecuciones diversas) y ofrecer una gran cantidad de combinacion
o posibilidades de resultados visuales o criteriales.

  • 🎯if else
  • 🧲switch case
  • 🍳for
    • for in
    • for of
    • for each
    • funcion arrow =>
  • 🥏while & Do While

Definida por su Sintaxis: 
                if(condicion == true){
                        *En esta acotacion, al igual que en las funciones
                            se ejecutará[las sentencias definidas]
                } else{
                         *Ejecucion de esta acotación siempre y cuando
                            la condicion inicial sea == false
                }
                Ejemplo de la ejecuciones
                var numero2 = 4;

                if (numero1  >=5) {
                    swal('Numero1 es mayor o igual que 5 ');
                } else if(numero2>=5) {
                    swal('Numero2 es mayor o igual que 5')
                }else {
                    swal('Ambos numeros son menor que 5')
                }
